⚠ Startup Error

ClawDesk could not start

The embedded server failed to launch. This usually happens because Node.js is not installed or is an incompatible version on this Mac.

How to fix

  1. Install Node.js 22 LTS — download and run the macOS installer from
    nodejs.org/en/download
  2. Quit and reopen ClawDesk — after installing Node.js, close this window and launch ClawDesk again from your Applications folder.
  3. If the error persists, check the Console.app for [clawdesk] log lines or reinstall ClawDesk.
